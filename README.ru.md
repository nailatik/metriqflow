Metriq Flow отслеживает ваши Telegram-каналы и VK-сообщества в едином дашборде, генерирует отчёты по запросу и доставляет их автоматически по расписанию — ежедневно, еженедельно или ежемесячно.

Создан для контент-команд и комьюнити-менеджеров, которым надоело собирать цифры вручную.

---

## Что внутри

### Аналитика Telegram
Подключите любой Telegram-канал, добавив бота администратором. Metriq Flow собирает:

- **Просмотры, реакции, пересылки, комментарии** на каждый пост — с бейджами роста относительно предыдущего периода
- **Engagement rate** за 24ч / 7д / 30д / всё время
- **Тепловая карта лучшего времени публикаций** — средние просмотры по часу и дню недели
- **Топ постов** — кликабельный список, отсортированный по просмотрам за выбранный период
- **График просмотров** — area chart с дневной детализацией
- **Авто-обновление** каждые 10 минут; новые каналы начинают отображать статистику в течение ~10 минут после добавления

### Аналитика VK
Подключите любое публичное VK-сообщество, вставив ссылку или screen name:

- **Охват, просмотры, посетители** — агрегированная статистика стены
- **Лайки, комментарии, репосты** по каждому посту
- **Engagement rate** и дельты роста
- **Топ постов**, **тепловая карта** и **график охвата**
- Периоды: 24ч / 7д / 30д / всё время

> Внутренняя аудиторная статистика VK (демография, источники трафика) закрыта для внешних приложений — Metriq Flow показывает только то, что доступно через публичный API VK.

### Единый дашборд
Вкладка **Все** объединяет Telegram и VK в один сводный экран: общее число подписчиков, совокупные просмотры и посты, разбивка по платформам. Один взгляд вместо двух вкладок.

### Отчёты по запросу
Сгенерируйте отчёт по любому каналу или сообществу в любой момент. Выберите:

- **Источник данных** — Telegram, VK или оба
- **Формат** — CSV (рекомендуется), PDF, XML
- **Период** — 24ч, 7 дней, 30 дней или всё время

CSV-отчёт включает: дату, ID поста, просмотры, реакции, пересылки, комментарии, engagement rate, наличие медиа и превью текста.

### Автоматические отчёты
Настройте расписание один раз — получайте отчёты бессрочно.

| Параметр | Варианты |
|---|---|
| Источник | Telegram / VK / Все |
| Формат | CSV / PDF / XML |
| Частота | Ежедневно / Еженедельно / Ежемесячно |
| Время отправки | Любой час (0–23), часовой пояс определяется автоматически |
| Канал доставки | DM в Telegram-боте, email или оба |

Расписания можно ставить на паузу, редактировать и переключать прямо из веб-приложения — или управлять ими из Telegram-бота.

### Telegram-бот
Привяжите аккаунт Metriq Flow к боту одноразовым токеном. Из Telegram вы можете:

- **Сгенерировать CSV-отчёт** — выбрать канал → выбрать период → получить файл за секунды
- **Управлять авторепортами** — просматривать все расписания, включать/отключать, переключать доставку в Telegram
- Получать плановые отчёты автоматически в личные сообщения

## Как это работает

**Шаг 1 — Подключите аккаунты**
Перейдите в **Интеграции**. Для Telegram: добавьте бота администратором в канал и вставьте токен привязки. Для VK: введите ссылку на сообщество или screen name.

**Шаг 2 — Аналитика появляется автоматически**
Metriq Flow начинает собирать данные сразу. Статистика Telegram обновляется каждые 10 минут; исторические посты VK импортируются при первом подключении.

**Шаг 3 — Смотрите инсайты, генерируйте отчёты**
Изучайте чарты и тепловые карты во вкладке Аналитика. Нажмите **Создать отчёт** для разовой выгрузки. Настройте расписание в разделе **Отчёты → Автоматические отчёты**, чтобы данные приходили сами.
